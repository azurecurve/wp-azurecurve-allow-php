== Description ==
Small, lightweight, plugin which allows PHP to be used on pages and posts; combine with <a href='http://wordpress.org/plugins/azurecurve-shortcodes-in-widgets/'>azurecurve Shortcodes In Widgets</a> to use PHP in widgets.

Use the [php][/php] shortcode in posts, pages or other post types to have the enclosed PHP code executed.

This plugin is multi-site compatible.

== Installation ==
To install the plugin copy the <em>azurcurve-allow-php</em> folder into your plug-in directory and activate it.
